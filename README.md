# Password Checker

Ziel dieses Projekts ist es, ein einfaches Java-Programm zu entwickeln, das vom Benutzer ein Passwort entgegennimmt und dessen Stärke bewertet.

## Ziel
- Benutzer gibt ein Passwort ein.
- Passwort muss ***mindestens 8 Zeichen*** und ***maximal 20 Zeichen*** lang sein.
- Es dürfen ***nur Buchstaben und Zahlen*** verwendet werden.
- Das Programm bewertet das Passwort als ***schwach***, ***mittel*** oder ***stark***.
- Bei einem schwachen Passwort muss der Benutzer so lange ein neues eingeben, bis mindestens ***mittel*** erreicht wird.
